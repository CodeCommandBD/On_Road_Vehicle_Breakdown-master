# Admin Dashboard Guide

The Admin Dashboard is the central command center for the On-Road Vehicle Breakdown platform. It allows administrators to manage users, garages, bookings, and view system-wide analytics.

## Accessing the Dashboard

Log in with an administrator account and navigate to `/admin/dashboard`.

## Key Features

### 1. Dashboard Overview

- **Real-time Stats**: View total users, active garages, today's bookings, and total revenue.
- **Recent Activity**: See the latest signups and bookings.
- **Revenue Chart**: Visual graph of income over time.

### 2. User Management

- Navigate to **Users** in the sidebar.
- **List Users**: View all registered car owners.
- **Search**: Find users by name or email.
- **Status Control**: Active/Suspend users if necessary.

### 3. Garage Management

- Navigate to **Garages**.
- **Pending Approvals**: Review new garage applications.
- **Verification**: Verify garage documents and approve/reject them.
- **Monitor Performance**: See which garages are performing best.

### 4. Bookings & Payments

- **Bookings**: View all service requests, their status (Pending, In Progress, Completed), and details.
- **Payments**: Track all transactions, including successful payments and refunds.

### 5. Analytics & Reports

- **Analytics**: Deep dive into platform usage, conversion rates, and financial reports.
- **Export**: Download data as PDF or CSV for offline analysis.

## Settings

- **System Settings**: Configure platform-wide variables (e.g., commission rates, default coordinates).
- **Service Categories**: Add or remove vehicle service types available to users.
